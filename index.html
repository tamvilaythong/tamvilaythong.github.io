Bear Maps

Bear Maps (Java)

Google Maps inspired web-based routing application. I implemented the back end for the mapping and routing of Berkeley by using existing front end and OpenStreetMap mapping data. Developed map rastering, zoom functionality, and clicking for location selection. I used a SAX parser with an OSM XML data file to build a graph representation of the Berkeley area, and used algorithm with the graph representation to implement shortest-path routing.

Developed and Designed by Tamara Vilaythong
